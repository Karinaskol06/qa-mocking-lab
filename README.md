# Лабораторна робота 10: Unit-тести з mocking та CI/CD

## Опис проєкту
Проєкт демонструє практичне застосування unit-тестування з використанням mocking та налаштування CI/CD за допомогою GitHub Actions.

## Швидкий старт

### Передумови
- Node.js 18+ (LTS)
- npm 9+
- Git

### Встановлення

```bash
# Клонувати репозиторій
git clone https://github.com/Karinaskol06/qa-mocking-lab.git
cd qa-mocking-lab

# Встановити залежності
npm install
```
### Запуск тестів
```bash npm test```	Запустити всі тести

```bash npm run test:unit```	Запустити unit-тести

```bash npm run test:coverage```	Запустити тести з покриттям

```bash npm run test:ci```	Запустити тести в CI режимі

```bash npm run lint```	Запустити лінтер
